This node intercepts requests and calculates token volume before routing.

Currently, the logic is set as follows:

  • If tokens < 2048: Route to local model (Ollama).
  • If tokens >= 2048: Redirect to cloud fallback chain (OpenAI -> Anthropic).

You can modify these thresholds in the source code.
